Pinko

Welcome to Pinko

Pinko is a distributed expression engine targetting Azure Cloud. The project is written in C# and F# (expression engine). The overall architecture is open source. The F# expression engine is not open source. The open source project contains a mock expression engine for sample purposes only.

Pinko is in early stages of development (Aug 2012) and not ccompleted at this point. You can set up a watch on our project to keep track of our progress.

RoadMap

The goal is to create a processing engine that processes user provided formulas in realtime across multiple Azure worker roles.

Pinko delivers real-time datafeed via SignalR. Initial version will include some sample formulas, but as the project grows it will include a full featured Windows 8 front end with Telerik controls.

Sample Expression

Think of Pinko as an Excel formula processor available in the cloud. A couple of sample expressions follow:

=RForm("Symbol", "IBM", "Price.Bid", "Reuters") * 2

Pinko monitors the IBM price bid. Every time a new Price.Bid arrives, the formula is recalculated and streamed into the client.

=RHist("Symbol", "IBM", "Price.Bid", "Reuters", "Min", 360) - RHist("Symbol", "IBM", "Price.Ask", "Reuters", "Min", 360)

RHist() retrieves 360 minutes calculates the Bid/Ask spread. Every time a new tick arrives, the formula is recalculated.

The Cool Stuff

So we have just another expression engine, but imagine creating a historical charts on demand with your own formulas, not pre-canned averages or spreads like other systems.

Pinko allows users to create formulas for trade ideas, color grids, ladders, historical charts, and any other analysis graphics.
